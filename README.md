# QuarcCode++

![Quarc](https://i.imgur.com/4IDI5pR.png)

 A minimalistic code editor written in C++ using ImGui and DirectX9
 
### Functional
- Displays the syntax of some languages. Currently Supported:
  - ะก/C++
  - LUA
  - SQL
  - GLSL
